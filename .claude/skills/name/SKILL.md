# Задача
Написать сопроводительное письмо от лица Андрея Михайличенко под конкретную вакансию.

# Алгоритм
1. Прочитай вакансию полностью
2. Найди что совпадает с опытом Андрея
3. Найди где пробелы — они будут упомянуты честно
4. Напиши письмо по структуре ниже
5. Проверь по чеклисту

# Структура письма
**Строка 1:** Почему эта вакансия — одна конкретная причина, не общая
**Строки 2-3:** Что релевантного есть — максимум 2-3 факта с цифрами
**Строка 4:** Честно про пробел если есть — одно предложение
**Строка 5:** Ссылка на сайт + зарплата

Итого: 5-7 предложений. Не больше.

# Тон и стиль
- Спокойный, уверенный, без пафоса
- Живой язык, не канцелярит
- От первого лица
- Без вводных слов: «безусловно», «несомненно», «хотел бы отметить»
- Без самооценок: «я ответственный», «люблю развиваться», «командный игрок»
- Пробелы признавать прямо, без оправданий

# Стоп-конструкции — никогда
- «не X, а Y»
- «я готов к новым вызовам»
- «хочу стать частью вашей команды»
- «меня привлекает ваша компания потому что»
- Любые клише которые мог написать любой другой кандидат
